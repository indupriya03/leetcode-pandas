# Merge & Join Operations (Pandas)

This folder contains **LeetCode Pandas problems** that focus on **merging tables** using  
`merge`, `join`, and `concat`.

These problems demonstrate how **SQL JOIN concepts** can be implemented using **Pandas DataFrames**.

---

## Concepts Covered
- LEFT JOIN
- INNER JOIN
- RIGHT JOIN
- Handling NULL / NaN values
- Combining multiple DataFrames
